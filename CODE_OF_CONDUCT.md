# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1, as its Code of
Conduct. The full text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: we want a friendly, respectful, and harassment-free community for
everyone. Be kind, assume good intent, and keep discussion constructive.

## Reporting

If you experience or witness unacceptable behavior, please report it privately
to the maintainer at **vaikar.abhijeet@gmail.com**. Reports will be reviewed and
handled confidentially.
